E-Commerce App (React Native + Expo)

A modern e-commerce mobile application built using Expo, React Native, Context API (Auth & Cart), Axios API integration, Formik, Yup, and modular UI components.

📌 Features

🔐 User Authentication

🛒 Cart Management

🛍️ Product Listing & Details

➕ Add Product Form (with Image Picker)

🎨 Beautiful Gradient UI

📂 Scalable Folder Structure (Modular components)

Requirements

Node.js & npm/yarn

Expo CLI or npx expo

Android Studio (for Android) / Xcode (for iOS on macOS)

JSON Server (for mock backend)

git clone <repository-url>
cd <project-folder>
npm install   # or yarn


npm install -g json-server
json-server --watch db.json --port 3000


npx expo start
